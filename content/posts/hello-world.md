Welcome to my new blog, built with [Hugo](https://gohugo.io/) and the
[LoveIt](https://hugoloveit.com/) theme, and hosted on GitHub Pages.

<!--more-->

## Why this blog

This is a placeholder first post. I'll write about whatever I'm working on,
learning, or thinking about.

## How it's built

- **Static site generator:** Hugo (extended)
- **Theme:** LoveIt
- **Hosting:** GitHub Pages, deployed automatically via GitHub Actions

## Writing a new post

Create a new post from the project root:

```bash
hugo new posts/my-next-post.md
```

Edit the Markdown, set `draft: false`, commit, and push — the site rebuilds
and deploys on its own.

Thanks for reading!
